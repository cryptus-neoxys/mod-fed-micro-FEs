# mod-fed-micro-FEs

This project utilises **Module Federation** to build *demo* Micro-Frontend Applications using React and Solid-js.

## To run

### Requirements

1. node
2. npm

### Running the apps

1. ```git clone https://github.com/cryptus-neoxys/mod-fed-micro-FEs```
2. ```cd mod-fed-micro-FEs```
3. Setup **remote**
    1. ```cd remote```
    2. ```npm i```
    3. ```npm start```
4. Setup **host**
    1. ```cd host```
    2. ```npm i```
    3. ```npm start```
5. Setup **solid-host**
    1. ```cd solid-host```
    2. ```npm i```
    3. ```npm start```
